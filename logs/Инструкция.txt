Для начала Вам потребуется создать таблицу в базе данных.
Создаём таблицу с двумя столбцами и даём ей имя (К примеру logs).
Названия столбцов должны быть следующими:
Первый столбец - datatimes (Тип - timestamp; По умолчанию - CURRENT_TIME; Артибулы - on update CURRENT_TIMESTAMP)
Второй столбец - string (Тип - VARCHAR; Длина - 150; Сравнение - cp1251_genera)
Сохраняем. Для проверки можно создать любую строку.
Далее открываем файл pwn и копируем его содержимое.
Открываем ваш мод, переходим к стокам и вставляем туда текст.
Вписываем вашу переменную подключения базы данныхи название таблицы с логами.
Компилируем.
Далее открываем файл logi.php с помощью редактора.
Вписываем данные БАЗЫ ДАННЫХ, название таблицы с логами.
Сохраняем и заливаем на хостинг для сайтов.
Переходим по домену и вуаля! Всё работает!
